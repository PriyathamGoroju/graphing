# Graph Plotting
This project is a web application that allows users to plot and analyze mathematical curves. It provides a user-friendly interface to enter mathematical expressions and visualize their corresponding curves using the Chart.js library.

## Usage
* Open the `index.html` file in a web browser.
* Enter a mathematical expression in the input field labeled `Enter`.
* Click the `Plot` button to generate the graph of the entered expression.
* The plotted curve will be displayed on the canvas area below the input field.
* Click on any point on the curve to view the tangent and normal equations at that point.

## Installation
No installation is required. Simply download the project files and open the `index.html` file in a web browser.

## Results
The application allows users to plot curves based on the entered mathematical expressions. It provides visual representations of the curves and displays the equations of tangents and normals at specific points on the curve.
